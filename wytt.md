# 网易新闻头条

> 获取 网易新闻头条 数据

## 接口参数

请求地址：`/api/new/toutiao`

请求方式: `get` `post`

请求参数

| 请求参数 | 必选 | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| `start` | 否 | int | 起始页，默认 `1` |
| `num` | 否 | int | 获取数量，默认10 |
