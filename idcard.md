# 身份证信息查询

> 可查询中国大陆居民身份证，港澳台居民身份证信息。可查询身份证的 **出生年月**，**居住地** ，**生日信息**
>
> **星座** ， **生肖**  **年龄** 等信息，支持 15 位和 18 位身份证

## 接口参数

请求地址：`/api/idcard`

请求方法：`get`  `post`

请求参数：

| 参数名称 | 必选 | 类型     | 说明 |
| -------- | ---- | -------- | ---- |
| `id`     | 是   | `string` | 要查询的身份证，支持 15 位和 18 位身份证,支持中大陆，港澳台身份证     |
